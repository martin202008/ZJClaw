# 商业招商 (Business Recruitment)

## 公司背景关联

宗靖商管在管5个商业综合体+1个专业卖场，总面积超44万㎡。招商工作需结合各项目定位：
- 苏州国际家居建材中心（21万㎡）：家居建材全业态
- 中翔时尚广场（7.2万㎡）：金融办公教育餐饮一体
- 中翔黄埭广场（5.36万㎡）：全生活商业矩阵
- 甪直中翔广场（4.2万㎡）：区域商业地标
- 周庄中翔广场（6.6万㎡）：辐射50万人口

## 技能描述

商业招商技能帮助商业地产项目完成品牌招商、租户管理和合同谈判等核心工作。

## 核心能力

### 1. 商业定位分析
- 目标客群分析
- 业态组合规划
- 品牌层级定位
- 租金定价策略

### 2. 招商策略制定
- 品牌库建设与维护
- 招商推广计划
- 渠道资源整合
- 招商节点把控

### 3. 租户评估与管理
- 租户资质审核
- 品牌适配度评估
- 经营能力分析
- 风险等级评定

### 4. 商务谈判支持
- 谈判策略建议
- 合同条款模板
- 租金方案设计
- 优惠政策制定

### 5. 合同管理
- 合同模板库管理
- 合同审批流程
- 履约跟踪监控
- 租金收缴管理

## 使用场景

### 招商计划制定
```
帮我制定XX项目的年度招商计划，包括：
- 招商目标设定
- 品牌落位规划
- 时间节点安排
- 预算资源配置
```

### 租户评估报告
```
对以下品牌进行入驻评估：
- 品牌名称：[品牌]
- 业态类型：[类型]
- 经营面积：[面积]
- 给出入驻建议和风险提示
```

### 合同条款审查
```
帮我审查租赁合同，重点关注：
- 租金条款
- 递增机制
- 免租期安排
- 违约责任
```

## 关键指标 (KPIs)

| 指标 | 说明 | 目标值参考 |
|------|------|-----------|
| 出租率 | 已出租面积/总可租面积 | ≥95% |
| 租金收缴率 | 实收租金/应收租金 | ≥98% |
| 品牌更替率 | 更替品牌数/总品牌数 | ≤10%/年 |
| 招商周期 | 从意向到签约平均天数 | ≤30天 |
| 首年租金坪效 | 首年租金/租赁面积 | 根据市场定价 |

## 行业最佳实践

### 品牌分级管理
- **A类品牌**：主力店、旗舰店，提供最优位置和最长免租期
- **B类品牌**：次主力店、知名连锁，标准招商政策
- **C类品牌**：补充业态，灵活调整招商条件
- **D类品牌**：应急填充，控制签约年限

### 招商节奏把控
1. 预招商期（开业前12-6个月）：主力店招商
2. 主力招商期（开业前6-3个月）：次主力店及重点品牌
3. 集中招商期（开业前3-1个月）：全面招商，补位招商
4. 开业冲刺期（开业前1个月）：快速填充，确保开业率
