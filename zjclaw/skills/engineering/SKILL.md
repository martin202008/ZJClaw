# 工程物管 (Engineering & Property Management)

## 公司背景关联

宗靖商管在管项目涵盖大型商业综合体和专业卖场，工程物管重点：
- **苏州国际家居建材中心**（21万㎡）：大型设备多，消防要求高
- **中翔时尚广场**（7.2万㎡）：办公+商业，电梯空调系统复杂
- **中翔黄埭广场**（5.36万㎡）：社区商业，能耗管控重点
- **甪直/周庄中翔广场**：设备老化风险，定期维护关键

## 技能描述

工程物管技能帮助商业地产项目实现设备设施管理、安全管理和环境维护的规范化、标准化。

## 核心能力

### 1. 设备设施管理
- 设备台账管理
- 维保计划制定
- 故障应急处理
- 设备更新改造

### 2. 安全管理
- 消防安全管控
- 治安安全管理
- 特种设备安全
- 应急预案编制

### 3. 环境管理
- 环境卫生标准
- 绿化养护管理
- 垃圾分类处理
- 空气质量监控

### 4. 能耗管理
- 能源统计分析
- 节能降耗措施
- 设备能效优化
- 成本控制分析

### 5. 装修管理
- 装修审批流程
- 施工现场管理
- 隐蔽工程验收
- 装修档案管理

## 设备管理体系

### 设备分级管理
| 级别 | 定义 | 巡检周期 | 维保周期 |
|------|------|----------|----------|
| A级 | 特种设备/核心设备 | 每日 | 按厂家要求 |
| B级 | 重要生产设备 | 每周 | 季度 |
| C级 | 一般设备 | 每月 | 半年度 |

### 设备档案管理
```
设备档案内容：
├── 基本信息（型号/厂商/参数）
├── 安装验收资料
├── 运行记录
├── 维保记录
├── 故障记录
└── 改造记录
```

## 安全管理体系

### 消防安全检查
```
日常检查项：
□ 消防通道畅通
□ 灭火器有效期
□ 消防栓可用性
□ 烟感/温感测试
□ 应急照明/疏散指示
□ 消防联动测试
□ 消防值班记录
```

### 安全检查分类
| 类型 | 周期 | 组织者 | 参与者 |
|------|------|--------|--------|
| 日检 | 每日 | 安保主管 | 保安员 |
| 周检 | 每周 | 安全经理 | 相关专业 |
| 月检 | 每月 | 项目总 | 全体管理人员 |
| 季检 | 每季 | 区域负责人 | 专业机构 |
| 年检 | 每年 | 集团安全部 | 第三方检测 |

## 应急预案

### 消防应急流程
```
发现火情 → 确认报警 → 启动预案 → 人员疏散
     ↓
灭火处置 → 医疗救护 → 现场保护 → 事故调查
```

### 电梯困人应急
```
安抚被困人员 → 确认位置 → 通知维保 → 实施救援
     ↓
安全放出 → 设备检修 → 故障分析 → 预防措施
```

### 停电应急
```
启动备用电源 → 保障安全 → 通知租户 → 快速抢修
     ↓
恢复供电 → 设备检查 → 原因分析 → 预防措施
```

## 能耗管理

### 能耗构成分析
```
商业建筑能耗分布（参考）：
├── 暖通空调：40-50%
├── 照明系统：20-25%
├── 电梯系统：10-15%
├── 其他设备：15-20%
```

### 节能措施
1. 设备升级（LED灯/变频空调）
2. 智能控制（BA系统优化）
3. 行为节能（定时开关/温度设定）
4. 余热回收
5. 新能源应用

## 装修管理流程

### 装修审批节点
```
申请 → 初审 → 复审 → 缴纳费用 → 办理证件 → 施工 → 验收
```

### 装修检查重点
- 消防改造合规性
- 结构改动安全性
- 油烟排放处理
- 噪音施工时间
- 垃圾清运规范
