# 数字营销 (Digital Marketing)

## 公司背景关联

宗靖商管定位城市商业服务运营商创新者，数字化营销重点：
- **苏州国际家居建材中心**：高客单、低频消费，需精准推送
- **中翔时尚广场**：年轻客群为主，种草+打卡营销
- **中翔黄埭广场**：社区型，邻里社群运营
- **甪直/周庄中翔广场**：本地+旅游双客群，需差异化触达

## 技能描述

数字营销技能帮助商业地产项目实现线上线下融合营销、私域流量运营和数字化转型升级。

## 核心能力

### 1. 营销策略规划
- 年度营销日历制定
- 营销预算分配
- 活动档期规划
- 渠道策略设计

### 2. 私域流量运营
- 微信公众号运营
- 小程序运营
- 社群运营管理
- 企微客户运营

### 3. 社交媒体营销
- 抖音/小红书运营
- 微博/视频号运营
- KOL/KOC合作
- UGC内容激励

### 4. 数据分析与洞察
- 营销效果分析
- 用户画像分析
- 转化漏斗分析
- ROI评估

### 5. 会员体系运营
- 会员等级设计
- 积分体系运营
- 权益体系设计
- 会员唤醒策略

## 关键指标 (KPIs)

| 指标 | 说明 | 目标值参考 |
|------|------|-----------|
| 粉丝增长率 | 月新增粉丝/上月粉丝 | ≥5% |
| 内容互动率 | 互动数/曝光数 | ≥3% |
| 私域转化率 | 转化人数/触达人数 | ≥2% |
| 活动参与率 | 参与人数/目标人群 | ≥10% |
| 会员活跃度 | 月活跃会员/总会员 | ≥30% |
| 营销ROI | 活动收益/营销成本 | ≥3:1 |
