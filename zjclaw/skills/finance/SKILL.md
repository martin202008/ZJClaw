# 财务管理 (Financial Management)

## 公司背景关联

宗靖商管属于中翔控股集团，财务管理需符合集团管控要求：
- **轻资产运营**：输出管理服务收取管理费
- **多项目协同**：5个商业综合体+1个专业卖场统一管理
- **收入结构**：租金+多种经营+停车+其他收入
- **成本管控**：人工+能源+维护+管理费用

## 技能描述

财务管理技能帮助商业地产项目实现预算管控、成本分析、收益管理和财务合规。

## 核心能力

### 1. 预算管理
- 年度预算编制
- 预算执行监控
- 预算调整管理
- 预算差异分析

### 2. 成本管理
- 成本预算编制
- 成本核算分析
- 成本控制措施
- 成本优化建议

### 3. 收入管理
- 租金收入管理
- 多种经营收入
- 停车场收入
- 其他收入管理

### 4. 资金管理
- 资金计划编制
- 应收账款管理
- 应付账款管理
- 资金使用效率

### 5. 财务分析
- 经营指标分析
- 财务比率分析
- 趋势分析预测
- 投资回报分析

## 关键指标 (KPIs)

| 指标 | 说明 | 目标值参考 |
|------|------|-----------|
| 租金收缴率 | 实收/应收租金 | ≥98% |
| 预算准确率 | 实际/预算偏差 | ≤±5% |
| 成本控制率 | 实际成本/预算成本 | ≤100% |
| 资金周转率 | 收入/平均资金占用 | 越高越好 |
| 毛利率 | 毛利/收入 | 根据业态 |
| 净利率 | 净利润/收入 | 根据业态 |
| ROE | 净利润/净资产 | ≥8% |

```
启动会 → 数据收集 → 初稿编制 → 部门审核
```

## 收入管理

### 收入构成
```
商业地产收入分类：
├── 租金收入
│   ├── 固定租金
│   ├── 提成租金（销售额分成）
│   └── 推广费/物业费
├── 多种经营收入
│   ├── 中庭出租
│   ├── 广告位出租
│   └── 活动场地费
└── 其他收入
    ├── 停车场收入
    ├── 装修管理费
    └── 违约金/赔偿金
```

### 租金定价策略
| 业态 | 定价原则 | 递增机制 |
|------|----------|----------|
| 餐饮 | 面积×单价 | 3-5%/年 |
| 零售 | 面积×单价 | 3-5%/年 |
| 体验 | 面积×单价 | 2-3%/年 |
| 主力店 | 一事一议 | 个性化约定 |

## 成本管理

### 成本分类
```
运营成本构成：
├── 固定成本
│   ├── 人工成本
│   ├── 折旧摊销
│   └── 保险费用
├── 变动成本
│   ├── 能源费用
│   ├── 维修维护
│   └── 清洁绿化
└── 期间费用
    ├── 管理费用
    ├── 销售费用
    └── 财务费用
```

### 成本控制方法
1. **预算控制**：严格按预算执行，超支审批
2. **比价采购**：多供应商比价，降低采购成本
3. **集中采购**：统一采购标准，降低单位成本
4. **效能分析**：分析成本效益，优化资源配置
5. **流程优化**：简化流程，减少不必要支出

## 税务管理

### 主要税种
| 税种 | 税率 | 纳税主体 |
|------|------|----------|
| 增值税 | 6%/9%/13% | 纳税人 |
| 企业所得税 | 25% | 企业 |
| 房产税 | 1.2%/12% | 产权人 |
| 土地使用税 | 定额 | 土地使用权人 |
| 印花税 | 0.1%等 | 合同双方 |

### 税务筹划要点
- 充分利用税收优惠
- 合理安排业务结构
- 规范成本费用核算
- 做好关联交易定价

## 风险控制

### 财务风险识别
- 资金流动性风险
- 应收账款坏账风险
- 成本超支风险
- 税务合规风险

### 内部控制要点
- 不相容职务分离
- 审批权限分级
- 定期对账盘点
- 内部审计监督
